# Palette: Online shop written with React and Tailwind
## Getting Started
1. `git clone https://github.com/MinooHadi/OnlineShop-Palette.git`
2. `cd OnlineShop-Palette`
3. `npm install`
4. `npm start`

**Note:** To run this project you need to run backend on port `3001`: 
5. `cd backend`
6. `npm install`
7. `npm run backend`
